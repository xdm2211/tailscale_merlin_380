Koolshare Asuswrt-Merlin Tailscale Changelog for Merlin 380 armv5 platform
===========================================

v1.0.0
   - add Tailscale插件上线，基于wiregurad协议的零配置内网穿透安全组网工具！
v1.0.1
   - 缩小体积，采用 combined 编译了二进制文件
      CGO_ENABLED=0 GOOS=linux GOARCH=arm GOARM=5 \
      go build -o tailscale.combined \
      -tags "ts_include_cli,ts_omit_aws,ts_omit_bird,ts_omit_tap,ts_omit_kube" \
      -ldflags="-s -w" \
      ./cmd/tailscaled   
   - 修复了一些小的逻辑错误   
v1.0.2
   - 增加了私有化部署选项
v1.0.3
   - Added minimal IPv6 support；
v1.0.4
   - 优化了可执行程序的内存回收的参数；   
v1.0.5
   - 修复了“客户端”角色无法连接到其他节点的问题；  
   - 测试后发现加上内存回收的参数会影响传输速度，在没有找到合适办法前先撤回这个改动； 
v1.0.6
   - 更新登录策略，若 state 文件存在，优先用 state（不带 --authkey）
